# 📝 Ian위키 웹 엔진

Ian위키 웹 엔진의 일부 공개 저장소입니다.

Ian위키는 PHP 기반으로 제작된 개인 위키 시스템으로,
문서 작성, 검색, 사용자 관리 등 위키 서비스에 필요한 기능을 구현하는 것을 목표로 합니다.

> ⚠️ 보안상의 이유로 전체 소스 코드는 공개하지 않습니다.
> 현재 저장소에는 공개 가능한 일부 코드와 디자인 관련 파일만 포함되어 있습니다.

---

## ✨ 주요 기능

현재 Ian위키에는 다음과 같은 기능이 구현되어 있습니다.

- 📄 문서 작성 및 관리
- 🔎 문서 검색 기능
- 👤 회원 시스템
- 🔐 세션 기반 로그인 관리
- 🎨 반응형 웹 디자인
- 🏠 위키 메인 페이지
- ⚙️ PHP 기반 웹 엔진 구조

---

## 🛠️ 사용 기술

| 기술 | 용도 |
|---|---|
| PHP | 서버 사이드 로직 |
| HTML | 웹 페이지 구조 |
| CSS | UI 디자인 |
| JavaScript | 웹 기능 확장 |
| MySQL / 데이터 저장 시스템 | 문서 및 사용자 데이터 관리 |
